# ConstructorPratik

Bu klasor ConstructorPratik adli .NET pratik calismalarini icerir.

## Calistirma
- Visual Studio ile acmak icin:
  - ConstructorPratik/ConstructorPratik.sln
  - (istege bagli) ConsoleApp1/ConsoleApp1.sln
- Komut satirindan calistirmak icin ornek:
  ```bash
  cd "ConstructorPratik"
  dotnet restore "ConstructorPratik/ConstructorPratik.sln"
  dotnet run --project "ConstructorPratik/ConstructorPratik/ConstructorPratik.csproj"
  ```

## Onemli Dosyalar
- ConstructorPratik/ConstructorPratik.sln
- ConstructorPratik/ConstructorPratik/ConstructorPratik.csproj
- ConsoleApp1/ConsoleApp1.sln (istege bagli)
